# hse_itogovoe_zadanie_python

## Описание
Скрипт анализирует логи Suricata и получает данные об уязвимостях из Vulners API, выявляет подозрительные IP, имитирует блокировку и формирует отчет с графиками.

## Источники данных
- Логи Suricata (alerts-only.json)
- Vulners API (критические уязвимости CVSS >= 7.0)

## Функции
- Анализ логов и статистика угроз
- Проверка IP через Vulners API
- Блокировка подозрительных IP
- Отчет в CSV и JSON
- Визуализация в PNG
